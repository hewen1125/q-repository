## 🚀 部署方式

### 选项1：GitHub Pages（推荐）
1. Fork 项目到你的 GitHub 账户
2. 在仓库设置中启用 GitHub Pages
3. 选择 "GitHub Actions" 作为源
4. 修改 `_config.yml` 中的配置
5. 推送代码，自动部署

### 选项2：本地运行
```bash
bundle install
bundle exec jekyll serve
# 访问 http://localhost:4000
```
